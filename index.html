Note 1: HTML Overview

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using a system of tags and attributes.

  • HTML is the skeleton of a web page.
  • It consists of elements, attributes, and content.
  • HTML works in combination with CSS (styling) and JavaScript (functionality) to build modern websites.

Syntax:

<html> <head> <body> ... </body> </html>

Example:

<html>
    <head></head>
    <body>Hello, world!</body>
</html>

Note 2: HTML Elements

HTML documents are made up of a variety of elements. An element typically consists of an opening tag, content, and a closing tag.

  • <html> - The root element of an HTML document.
  • <head> - Contains meta-information, links to stylesheets, and other resources.
  • <body> - Contains the visible content of the web page.
  • <h1> to <h6> - Header tags used to define headings (h1 is the largest, h6 the smallest).
  • <p> - Defines a paragraph of text.
  • <a> - Defines a hyperlink.
  • <ul>, <ol> - Unordered and ordered lists.
  • <img> - Defines an image.
  • <div> - A block-level container used for grouping and styling content.

Example:

<h1>Main Heading</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">Click Here</a>

Note 3: HTML Attributes

Attributes provide additional information about an HTML element. They are always written within the opening tag of an element.

  • id - Assigns a unique identifier to an element.
  • class - Assigns one or more class names to an element, useful for CSS and JavaScript.
  • href - Defines the URL for a hyperlink (used within the <a> tag).
  • src - Defines the source of an image (used within the <img> tag).
  • alt - Provides alternative text for an image if it cannot be displayed.
  • title - Provides additional information when hovering over an element (e.g., a tooltip).
  • style - Inline CSS styles for an element.

Syntax:

<element attribute="value"> ... </element>

Example:

<a href="https://example.com" title="Visit Example">Example Link</a>
<img src="image.jpg" alt="Description">

Note 4: Forms

Forms are used to collect input from users. The <form> element encapsulates the entire form structure.

  • <form> - The root element for creating forms.
  • <input> - Defines an input field for text, checkboxes, radio buttons, etc.
  • <label> - Associates a label with a form element.
  • <textarea> - Defines a multi-line text input field.
  • <select> - Creates a dropdown list of options.
  • <button> - Defines a clickable button.
  • method="get" vs. method="post" - Defines how form data is sent (GET appends data to the URL, POST sends it in the request body).

Example:

<form method="post" action="/submit">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

Note 5: Semantic Tags

Semantic HTML elements clearly describe their meaning in a human- and machine-readable way.

  • <header> - Represents the header of a section or page.
  • <footer> - Represents the footer of a section or page.
  • <article> - Represents a self-contained, independent piece of content.
  • <section> - Represents a section of content.
  • <nav> - Defines navigation links.
  • <aside> - Represents content aside from the main content (like a sidebar).
  • <main> - Represents the main content of the document.

Example:

<header>Header Content</header>
<nav>Navigation Links</nav>
<article>Article Content</article>
<footer>Footer Content</footer>

Note 6: Tables

Tables are used to display data in rows and columns. They are created using the <table> element.

  • <table> - The container element for the table.
  • <tr> - Defines a table row.
  • <th> - Defines a table header cell (bold and centered by default).
  • <td> - Defines a table data cell.
  • <thead>, <tfoot>, <tbody> - Grouping elements for the table's header, footer, and body.

Example:

<table>
    <thead>
        <tr><th>Header 1</th><th>Header 2</th></tr>
    </thead>
    <tbody>
        <tr><td>Data 1</td><td>Data 2</td></tr>
    </tbody>
</table>

Note 7: Multimedia

HTML allows for embedding multimedia such as audio and video into web pages.

  • <audio> - Embeds audio content.
  • <video> - Embeds video content.
  • <source> - Specifies multiple media resources for elements like <audio> and <video>.
  • <iframe> - Embeds another HTML page within the current page (commonly used for embedding videos or other content).

Example:

<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Your browser does not support the audio element.
</audio>
<video controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video element.
</video>
